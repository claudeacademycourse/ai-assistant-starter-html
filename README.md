 AI Assistant Starter

A friendly web design tutor chatbot. Ask it about fonts, colors, layouts, and more.

## How to run

1. Download `index.html`
2. Double-click it

That's it. No installs, no terminal, no npm. Just a browser and an internet connection.

## What it does

- Landing page with a "Start Chatting" button
- Chat interface with suggested prompts
- Instant AI-style responses about web design basics (fonts, colors, layouts, HTML/CSS, CTAs, and more)
- Fully private - nothing is sent to a server, everything runs in your browser

## Built with

- React 18 (loaded from CDN)
- Tailwind CSS (loaded from CDN)
- Babel Standalone (JSX in the browser, no build step)
- Zero dependencies to install
